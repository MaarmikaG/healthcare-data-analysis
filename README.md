# 🏥 Healthcare Data Analysis Dashboard (Power BI)

## 🔹 Overview

This project focuses on analyzing healthcare data using Power BI to uncover insights related to hospital operations, patient trends, and financial metrics. The dashboard is designed to be interactive and user-friendly, enabling detailed exploration of patient and hospital data.


## 🔹 Objectives

* Analyze patient data and hospital performance
* Track billing and insurance trends
* Evaluate patient distribution across diagnoses
* Improve visibility into hospital operations


## 🔹 Tools & Technologies

* **Power BI** – Data visualization and dashboard creation
* **Excel / CSV** – Dataset handling


## 🔹 Key Insights

* Variations in billing based on bed occupancy levels
* Differences between insurance coverage and actual billing
* Identification of high-volume diagnoses
* Performance insights based on patient feedback
